# Определение перспективного тарифа для телеком-компании

## Стек:
Python, Pandas, Numpy, Matplotlib/Seaborn, SciPy, Statsmodels
![Python](https://img.shields.io/badge/Python-3.9-blue?logo=python&logoColor=white)

## Структура проекта:
1. Изучение данных
2. Подготовка данных:
	- Приведение типов;
	- Реализация логики округления;
	- Создание сводного DataFrame.
3. Анализ данных
	- Дескриптивная статистика;
	- Визуализация распределения;
	- Расчет ключевых метрик.
4. Статистическая проверка гипотез:
   - Предварительные тесты;
   - Основной тест.
5. Выводы

## Методы и подходы:
- Обработка данных: бизнес-правила округления, денормализация, агрегация;
- Анализ данных: Стратифицированный анализ, EDA (Exploratory Data Analysis);
- Визуализация распределений;
- Статистика: проверка нормальности, проверка гомогенности дисперсий, t-тест Уэлча, размер эффекта (Cohen's d), анализ мощности, доверительные интервалы.

## В ходе проекта решил следующие задачи:
- Осуществил интеграцию разнородных источников;
- Привел данные к корректным типам;
- Создал целевые признаки;
- Провел анализ пользовательской базы, потребительского поведения;
- Провел структурный анализ выручки;
- В статистическом анализе выполнил:
   - проверку распределения на нормальность;
   - проверку гомогенности дисперсий;
   - оценку размера эффекта;
   - анализ статистической мощности;
   - рассчитал доверительные интервалы.
